== Description ==
SnazzyAdmin WP Admin Theme – Takes a fresh, elegant and snazzy approach to your WordPress website’s admin dashboard design. With SnazzyAdmin, reinvent the The look and feel of your wordpress admin dashboard. Add profile photo and cover images as per your personal choice.

Features:
* Add Profile photo to replace default gravatar image everywhere
* Apply cover Photo to your bashboard
* Fully Responsive
* Change color scheme

PS: You'll find customization options in "Users > Your Profile" Section

== Installation ==

Upload the snazzyadmin-wp-admin-theme folder to /wp-content/plugins/
Activate the plugin through the 'Plugins' menu in WordPress
Navigate to 'Users > Your Profile' in wp-admin.
